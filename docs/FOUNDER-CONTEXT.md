## Pop-Up Activation Niche

Distinct from weddings/private parties — requires different feature emphasis.

### Key Differences

| Wedding/Private Party | Pop-Up Activations (Agencies) |
|-----------------------|-------------------------------|
| Single event per account | Multiple simultaneous events |
| One location | Different locations, same day |
| Solo or small team check-in | Teams of staff per location |
| Host-focused experience | Client-facing reporting required |
| Aesthetic customization critical | Speed of deployment critical |

### Feature Implications

| Need | Current Support | Gap/Opportunity |
|------|----------------|-----------------|
| Multi-event management | Pro tier (unlimited events) | May need simultaneous event view |
| Real-time sync across devices | Planned | Critical for agency ops |
| Post-event data exports | Basic CSV? | Client-ready reporting (branded PDFs) |
| Rapid deployment | 2-minute setup | Template system for repeat clients |
| Location tracking | None? | Geo-tagged check-ins for multi-site |

### Potential "Agency" Tier

- Position between Pro ($29-49) and Business ($99-149)
- ~$79/month: Unlimited simultaneous events, team management, client reporting exports, template library
- Alternative: Usage-based for agencies (pay per activation, not monthly)

## Scanner Gun Replacement Narrative

### Hardware Pain Points I Witnessed

- Battery failures mid-event
- Bluetooth pairing drops
- Connectivity dependency (WiFi/4G)
- Single point of failure (one gun vs. multiple phones)
- No visual feedback for guests (feels unprofessional)

### Our Advantage

- Phone-as-scanner: everyone has one, no hardware to fail
- Offline-first: works in airplane mode (venue WiFi independence)
- Visual + audio feedback: satisfying for staff, professional for guests
- Multi-device redundancy: if one phone dies, grab another

### Messaging

- "Ditch the scanner gun" — explicit comparison
- Document reliability metrics vs. hardware (uptime, failure rate)
- Emphasize "staff already have the hardware in their pocket"
